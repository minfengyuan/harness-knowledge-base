# 菜谱分类参考

## 烹饪方式分类

| 分类 | 标签 | 判断依据 |
|------|------|----------|
| 小炒 | `#小炒` | 锅内快速翻炒，油温高，时间短（爆炒、干煸、滑炒） |
| 炖菜 | `#炖菜` | 长时间液体慢煮（炖、煲、焖、红烧） |
| 蒸菜 | `#蒸菜` | 蒸汽加热为主（清蒸、粉蒸） |
| 凉菜 | `#凉菜` | 冷食，拌/腌/卤后放凉（凉拌、卤味、白切） |
| 烤制 | `#烤制` | 烤箱或明火烤（烧烤、烘焙咸食） |
| 炸制 | `#炸制` | 油炸为主要手法（酥炸、干炸、软炸） |
| 煮制 | `#煮制` | 水煮或汤煮（水煮、白灼、涮） |

## 菜品类型分类

| 分类 | 标签 | 判断依据 |
|------|------|----------|
| 主食 | `#主食` | 米饭、面条、饼、馒头、饺子等碳水为主 |
| 汤羹 | `#汤羹` | 液体为主的菜品（汤、羹、粥） |
| 甜品 | `#甜品` | 甜味为主的食物（糕点、布丁、冰品） |
| 饮料 | `#饮料` | 饮品类（奶茶、果汁、茶饮、酒酿） |
| 酱料 | `#酱料` | 调味用途（酱、蘸料、复合调料） |

## 分类规则

1. 每道菜至少打一个**烹饪方式**标签和一个**菜品类型**标签
2. 如果兼有多种手法（如先炸后炖），取主要手法，可叠加标签
3. 标签放在标题行末尾，空格分隔
4. 无法确定时标注 `#其他`
